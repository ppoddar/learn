# Introduction 
simple node.js application to learn the environment

# Getting Started

On Mac:

   1. Checkout the project from Azure devops

       

   2.  deploy a node.js based web server on the VM. Public ip
        ```
        $ ./bin/deploy.sh
        ```

   3. open up browser page
      ```
         $ open http://$PUBLIC_IP/
      ```
